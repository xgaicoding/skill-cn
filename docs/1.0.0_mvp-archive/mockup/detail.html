这个 Skill 能做什么？

frontend-design 用于把 PRD/线稿快速变成可交付的 UI 方案：包含布局、组件拆分、视觉风格、交互细节与验收检查清单。

如果检测到 Markdown 里包含相对路径图片/链接（如 ./ / ../），MVP 会降级为“纯文本展示”，并提供 “前往 Source”。

快速开始

在终端中执行：

bashcodex --skill frontend-design

# 输入：你的 PRD / 页面目标 / 竞品参考
# 输出：页面结构 + 视觉规范 + 可访问性清单

建议的输出结构（示例）

模块 建议内容 验收点
Header Logo + 搜索 + 登录态 + 提交 CTA（区分主次） 悬浮不遮挡内容；键盘可聚焦
Banner 左品牌区；右精选实践轮播（箭头 + 圆点） 与筛选/搜索无关；3–6 条才展示
详情页 标题/标签/简介 + 仓库信息 + 操作区 + 实践区 + SKILL.md Run 置灰且提示；skill_id 可复制

注意事项

建议为卡片、按钮、输入框提供明显的 hover 与 focus-visible 状态；动画遵循 prefers-reduced-motion。
