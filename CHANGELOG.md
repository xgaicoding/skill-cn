# Changelog

## [1.2.0] - 2016-01-31
- 移动端上线，支持H5

## [1.1.0] - 2026-01-27
- 首页新增实践模式（`mode=practices`）：右侧吸附 Mode Dock，文章等高卡片网格，支持分类筛选/最热最新排序/分页，Header 搜索切换为“实践搜索”。

## [1.0.1] - 2026-01-25
- 支持单篇实践关联多个 skill（多对多基础）。

## [1.0.0] - 2026-01-23
- Skill Hub 中国是一个面向 Agent/AI 场景的 Skill 生态平台，核心目标是“聚合优质 Skill 与落地实践，让开发者更快构建可复用能力”。整体体验偏“轻量浏览 + 低门槛共建”，既能快速找答案，也能方便提交与分享。
